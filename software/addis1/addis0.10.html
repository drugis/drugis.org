Release history

Release 0.10

Release Date: 2010-06-21

We are happy to announce the release of ADDIS (Aggregate Data Drug Information System) version 0.10. In this release, we build upon meta-analysis (included since ADDIS v0.4) and network meta-analysis (included since ADDIS v0.8), to perform multi-criteria benefit-risk analysis of alternative treatment options. The benefit-risk method is based on SMAA, and implemented using the JSMAA software. It is being developed specifically for medical and regulatory decision making by our research group, see our publications for more information.

We include an example case (benefit-risk assessment of anti-depressants) with ADDIS v0.10. To try out benefit-risk analysis, load the included file data/hansen-analyses.xml. This file contains data we extracted from studies that were included in the Hansen et al., Annals of internal medicine, 143(6): 415-426, 2005 systematic review of second-generation antidepressants. We also included several example analyses, including a benefit-risk model, "Hansen BR".

Running ADDIS

Extract the addis-0.10.zip ZIP archive to a local directory. Then double-click the 'addis-0.10.jar' file. Now you should already have ADDIS 0.10 running with some provided example data.

If double-clicking the JAR file did not launch ADDIS, you are most likely lacking the Java run-time environment. You can get this from java.com.
